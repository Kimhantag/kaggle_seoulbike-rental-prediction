# kaggle_seoulbike-rental-prediction

참여 : 1명  
기간 : 1 week   
내용 : Data visualization 과목에서 개인으로 진행한 서울시 공공자전거 예측 모델링  
피드백 : 시간(Hour)에 대한 변수를 one-hot encoding을 통해 진행하고, 선형회귀모형을 통해 변수의 유의성을 판단하여 타 모델에 반영하는 작업을 고려하지 못함.
  
* 고려사항
1) 원핫인코딩된 변수 사용 : 원핫인코딩된 변수들이 트리모형에서도 중요한 특징이 될 수 있으므로, 이들을 그대로 사용하는 것이 유리할 수 있음.
2) 변수 선택 기준 : 선형회귀모형에서 유의미한 변수로 선택된 원핫인코딩된 변수들이 트리모형에서도 중요한 변수일 가능성이 높습니다. 그러나 트리모형에서는 원핫인코딩된 모든 변수를 다시 한 번 검토하여 중요한 변수를 선택해야 함.
3) 시간 변수를 라벨인코딩하는 경우 : 원핫인코딩된 변수들 대신 시간 변수를 라벨인코딩하여 사용하는 것도 가능함. 그러나 이 경우 선형회귀모형에서 유의미한 변수로 선택된 변수들과의 일관성을 유지하기 어려울 수 있음
